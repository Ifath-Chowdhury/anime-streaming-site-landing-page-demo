# Anime Streaming Site Landing Page Demo - AnimeSky

This is a demo for a landing page of an anime streaming site which is configured with Vercel Analytics (through a `script` tag), advanced routing with [Vercel Edge Middleware](https://vercel.com/docs/concepts/functions/edge-middleware), as well as some basic styles.
